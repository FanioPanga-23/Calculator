# Calculator

A small web-based calculator built with plain HTML, CSS and JavaScript.

**Core description:**

- Implements a basic calculator UI and logic: addition, subtraction, multiplication, division

**Files**

- [index.html](index.html) — main page and app container.
- [js/index.js](js/index.js) — application logic and event handling.
- [style/index.css](style/index.css) — styles and layout.
- [fonts/digital-7/](fonts/digital-7/) — optional 7-segment style font used by the display.

**How to run**

- Download or clone the repository and open `index.html` in a browser.
- Alternatively view the project on GitHub Pages if hosted (site URL or project pages).
